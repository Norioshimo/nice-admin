# Proyecto de Gestión con Nice Admin (React + Spring Boot)

Este proyecto consiste en la migración del template Nice Admin a un stack de React para el frontend y Spring Boot (Java 17) para el backend. El objetivo principal es implementar los CRUDs necesarios y mantener una arquitectura modular y escalable.

## Tecnologías utilizadas
### Frontend
- React 19
- React Router
- Axios (para llamadas a la API)
- Nice Admin Template (adaptado a React)

### Backend
- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- Base de datos: (indicar la que uses, por ejemplo MySQL o PostgreSQL)
- Maven o Gradle como gestor de dependencias

### Características
- Migración del template Nice Admin a React.
- Implementación de CRUDs completos para las entidades del sistema.
- Integración del frontend con el backend mediante API REST.
- Arquitectura escalable y modular para fácil mantenimiento.
- Gestión de rutas y navegación en React.
- Manejo de estados con hooks de React (useState, useEffect).
